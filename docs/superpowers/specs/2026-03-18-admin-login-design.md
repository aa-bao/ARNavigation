# Admin Login Design

**Goal:** Refresh the admin login page into a restrained institutional interface that feels credible for a hospital management system rather than a generic startup dashboard.

## Direction

Use a clean administrative aesthetic:
- Warm off-white background instead of gradient-heavy hero art
- Dark slate typography with a limited hospital-blue accent
- Serif-led heading paired with restrained UI text
- Light panel treatment instead of glossy glassmorphism

## Layout

- Split the page into a left institutional intro panel and a right login form panel
- Keep the composition asymmetric but calm, with generous spacing
- On small screens, collapse to a single-column card while preserving hierarchy

## Visual Language

- Minimal decorative elements inspired by hospital wayfinding: fine rules, small labels, section markers
- No neon glow, purple gradients, floating blobs, or cyber motifs
- Motion limited to subtle fade/slide entrance and focused hover states

## Interaction

- Preserve the existing login behavior and form validation
- Improve field affordance, button feedback, and loading state clarity
- Keep contrast strong and the interface easy to scan
