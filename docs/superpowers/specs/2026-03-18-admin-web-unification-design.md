# Admin Web Unification Design

**Goal:** Unify the admin web interface under the same restrained hospital-institutional aesthetic established by the login page.

## Scope

- Shared shell layout
- Location management page
- QR code batch generation page

## Direction

- Warm neutral background with controlled hospital-blue accents
- Structured page headers with subtle wayfinding cues
- Refined cards, tables, dialogs, and toolbars
- Chinese copy only, with corrupted text repaired

## UX Rules

- Keep existing data flows and actions unchanged
- Improve hierarchy and scanability rather than adding features
- Preserve responsive behavior for narrower screens
